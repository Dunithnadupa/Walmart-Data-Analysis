# 📊 Walmart Data Analysis Project

## 🚀 Overview

This project is an **end-to-end data analysis solution** designed to extract meaningful business insights from Walmart sales data. It demonstrates the complete workflow of a data analyst — from data collection to cleaning, transformation, and advanced SQL-based analysis.

The project focuses on solving **real-world business problems** using **Python and SQL**, making it ideal for showcasing practical data analytics skills.

---

## 🛠️ Tech Stack

* **Python** (Data Cleaning & Processing)
* **SQL** (MySQL & PostgreSQL for Analysis)
* **Libraries**: pandas, numpy, sqlalchemy, pymysql, psycopg2
* **Tools**: VS Code, Kaggle API

---

## 📌 Key Features

✔ End-to-end data analysis pipeline
✔ Data cleaning and preprocessing
✔ Feature engineering
✔ SQL-based business problem solving
✔ Cross-database implementation (MySQL & PostgreSQL)

## ⚙️ Project Workflow

### 1️⃣ Environment Setup

* Organized project structure using VS Code
* Configured Python and SQL environments

### 2️⃣ Data Acquisition

* Downloaded dataset using **Kaggle API**
* Stored data in the `data/` directory

### 3️⃣ Data Exploration

* Performed initial analysis using:

  * `.info()`
  * `.describe()`
  * `.head()`

### 4️⃣ Data Cleaning

* Removed duplicate records
* Handled missing values
* Fixed inconsistent data types
* Cleaned and formatted currency values

### 5️⃣ Feature Engineering

* Created new columns such as **Total Amount**
* Improved dataset for better SQL analysis

### 6️⃣ Database Integration

* Loaded cleaned data into:

  * MySQL
* Automated table creation using SQLAlchemy

### 7️⃣ SQL Analysis

Solved **9 key business problems**, including:

* Revenue trends by branch and category
* Best-selling product categories
* Sales by city, time, and payment method
* Customer buying patterns
* Profit margin analysis

## ⚡ Getting Started

### 1. Clone the Repository

```
git clone <your-repo-url>
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Setup Kaggle API

* Download `kaggle.json` from your Kaggle account
* Place it in the `.kaggle/` folder

### 4. Download Dataset

```
kaggle datasets download -d <dataset-path>
```
